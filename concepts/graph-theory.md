# 图论基础

## 定义

图论是研究**图（Graph）** 的数学学科——图由顶点（Vertex）和边（Edge）组成，是计算机科学中最通用的数据抽象之一。社交网络、地图导航、编译器依赖分析、知识图谱都以图为基础。

## 基本概念

| 概念 | 说明 |
|------|------|
| 顶点 (V) | 图中的节点 |
| 边 (E) | 节点间连接：有向/无向 |
| 度 (Degree) | 顶点连接的边数 |
| 路径 (Path) | 顶点序列，边连接相邻顶点 |
| 环 (Cycle) | 起点=终点的路径 |
| 连通分量 | 图中连通的最大子图 |
| 邻接矩阵 | `A[i][j] = 1` 表示边连接，O(1) 查询，O(V²) 空间 |
| 邻接表 | 每个顶点存储邻居列表，O(E) 空间，适合稀疏图 |

## 图的分类

| 类型 | 特点 | 应用 |
|------|------|------|
| 无向图 | 边无方向 | 社交网络 |
| 有向图 (Digraph) | 边有方向 | 网页链接、依赖关系 |
| 加权图 | 边带有权重 | 地图导航（距离） |
| DAG | 有向无环图 | 拓扑排序、编译器 |
| 二分图 | 顶点分两集合，只跨集合连边 | 匹配问题 |
| 树 | 连通无环图 | 文件系统、组织结构 |
| 完全图 | 每对顶点都相连 | 理论分析 |

## 核心算法

### 遍历
| 算法 | 数据结构 | 复杂度 | 用途 |
|------|----------|--------|------|
| BFS | 队列 | O(V+E) | 最短路径（无权图） |
| DFS | 栈/递归 | O(V+E) | 连通分量、拓扑排序、环检测 |

### 最短路径
| 算法 | 限制 | 复杂度 | 场景 |
|------|------|--------|------|
| Dijkstra | 边权 ≥ 0 | O((V+E) log V) | GPS 导航 |
| Bellman-Ford | 允许负权 | O(VE) | 检测负环 |
| Floyd-Warshall | 全源最短 | O(V³) | 所有点对 |
| A* | 启发式 | 取决于启发函数 | 游戏寻路 |

### 最小生成树
| 算法 | 数据结构 | 复杂度 |
|------|----------|--------|
| Kruskal | 并查集 | O(E log E) |
| Prim | 优先队列 | O(E log V) |

### 其他
- **拓扑排序 (Kahn / DFS)**：DAG 的线性排序，编译器依赖解析
- **强连通分量 (Kosaraju / Tarjan)**：有向图中互相可达的最大子图
- **二分图匹配 (Hungarian Algorithm)**：最大匹配
- **最大流 (Ford-Fulkerson / Edmonds-Karp)**：网络流
- **欧拉路径 / 哈密顿路径**：每条边一次 / 每个顶点一次

## 与计算机科学的联系

- **编译原理**：AST 是树，CFG 是图
- **网络**：路由算法是图的最短路径
- **数据库**：查询计划优化 → 图搜索
- **AI**：知识图谱、贝叶斯网络、神经网络拓扑
