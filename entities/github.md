# GitHub

## 概览

GitHub 是全球最大的代码托管和协作平台，2008 年上线，2018 年被 Microsoft 收购。基于 [[git]]，提供协作、CI/CD、包管理、AI 编程助手等完整开发者生态。

## 核心功能

| 功能 | 说明 |
|------|------|
| Repository | 代码仓库 |
| Pull Request | 代码审查和合并流程 |
| Issues | 任务/bug 跟踪 |
| Actions | CI/CD 工作流平台 |
| Codespaces | 云端开发环境（VS Code in browser） |
| Copilot | AI 编程助手（代码补全、聊天、Agent） |
| Pages | 静态网站托管 |
| Packages | 包注册表（npm, Docker, Maven 等） |

## GitHub Flow

```
main ← feature branch → PR review → merge
```

最简洁的工作流，适合大多数团队。

## 竞品

| 平台 | 特点 |
|------|------|
| GitLab | 内置 DevOps 全流程，可自托管 |
| Gitee | 国内平台，网速快 |
| Bitbucket | Atlassian 生态（Jira, Confluence） |
