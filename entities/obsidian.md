# Obsidian

## 概览

Obsidian 是基于 **本地 Markdown 文件** 的知识管理和笔记应用。由 Shida Li 和 Erica Xu 于 2020 年发布。核心哲学：你的笔记归你所有，存储在本地，永远不会被锁定。

## 核心特性

- **本地优先**：所有笔记都是标准 Markdown 文件
- **双向链接**：`[[wikilinks]]` 连接笔记，构建知识网络
- **关系图谱**：可视化笔记之间的链接
- **社区插件**：2000+ 插件扩展功能
- **Canvas**：无限画布白板

## 常用插件

| 插件 | 用途 |
|------|------|
| Obsidian Git | 自动 Git 备份同步 |
| Dataview | 用查询语言筛选/展示笔记 |
| Templater | 高级模板引擎 |
| Calendar | 日历视图 + 日记导航 |
| Excalidraw | 手绘图表 |
| Kanban | 看板视图 |

## Vault 同步方案

| 方案 | 特点 |
|------|------|
| Obsidian Sync | 官方方案，付费，端到端加密 |
| [[git]] + Obsidian Git 插件 | 免费，支持任意平台 |
| iCloud / OneDrive | 简单但可能冲突 |

## 局限

- 不是开源的（核心闭源）
- 移动端体验不及桌面端
- 没有内置协作编辑
- 大量插件可能导致启动慢
